# Project Deployment
This folder contains all the files used in the deployment process of this project.

### Deployment Stack:
- Cloud provider
  - GCP
- Model Storage
  - GCP VertexAI Model registry
  - Artifact registry
- Model serving
  - FastAPI
  - GCP VertexAI endpoints
- CI/CD
  - GitHub Actions
- Containerization
  - Docker
- Frontend
  - Firebase Hosting
