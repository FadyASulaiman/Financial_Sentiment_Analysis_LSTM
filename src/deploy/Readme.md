# Project Deployment
This folder contains all the files used in the deployment process of this project.

Deployment Stack:
- Cloud provider
  - GCP
- Model Storage
  - GCP VertexAI Model registry
  - Artifact registry
- Model serving
  - GCP VertexAI endpoints
- Frontend
  - Firebase Hosting
- CI/CD
  - GitHub Actions
- Pipelines
  - GCP VertexAI Pipelines
